#### So, how to contribute?

---

<ul>
  <Appear>
    <li>Pick a project</li>
    <li>Understand it</li>
    <li>Look for issues / think of a feature</li>
    <li>Discuss with other contributors</li>
    <li>Follow the Code of conduct &amp; contributions guidelines</li>
    <li>Create a Pull request</li>
    <li>Make sure tests pass and there are no conflicts</li>
    <li>After a review, fix bugs/suggestions if any</li>
    <li>You pull request (PR) will be merged</li>
  </Appear>
</ul>
